## 🎮 Мои проекты

### 🎭 Sarcasm Detection Service (NLP & MLOps)
**Промышленный сервис для определения сарказма в английских текстах**

* **Что сделано:** Проведен EDA, подготовка данных и сравнение 4 моделей. Выбрана **Logistic Regression** (F1: 0.82), как самая быстрая и точная для данного типа задач.
* **Стек:** `FastAPI`, `Scikit-learn`, `Docker`, `Joblib`.
* **Архитектура:** Применен модульный подход (ООП). Логика обработки вынесена в отдельный класс, сервис упакован в Docker-контейнер для легкого развертывания.
* **Интерфейс:** Реализован REST API с автоматической документацией Swagger.
